# Day 1 — Wallets & Transactions
**Date:** March 04, 2026

## What I Learned

### 1. What is a Crypto Wallet?
- A wallet does NOT store coins — it stores your **private key**
- Your coins live on the blockchain
- Wallet = key to access and sign transactions

### 2. Public Key vs Private Key
```
Private Key → only you have this → NEVER share
Public Key  → your wallet address → share freely
```

### 3. How a Transaction Works
1. You create a transaction (send X ETH to address Y)
2. You sign it with your private key
3. It gets broadcast to the network
4. Miners/validators confirm it
5. It's permanently recorded on the blockchain

### 4. Types of Wallets
- **EOA (Externally Owned Account)** — controlled by private key (your MetaMask)
- **Contract Account** — controlled by code (smart contract)

### 5. MetaMask Setup
- Installed MetaMask extension
- Connected to Ethereum Sepolia Testnet
- Got test ETH from faucet
- Sent first test transaction

## Key Takeaway
> Your private key IS your identity on-chain. Lose it = lose everything. Share it = lose everything.
