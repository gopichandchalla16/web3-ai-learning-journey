## What I Built

A complete **AI Wallet Analyser** that combines everything from Days 1–6:

- Connects to Ethereum via Web3.py (Day 5)
- Reads wallet balance live on-chain (Day 5)
- Gets transaction count (new today)
- Gets latest block info (new today)
- Generates a plain-English AI report (Day 6 concept)
- Classifies wallet activity automatically

**Input:** any Ethereum wallet address  
**Output:** full AI report with balance, tx count, activity level, plain-English summary
